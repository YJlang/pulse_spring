Base URL: http://localhost:8080/api

Last Updated: 2026-01-10

클릭하여 상세 내용을 확인하세요. 모두 펼치기 / 접기 기능은 우측 상단 버튼을 활용하세요.

POST /signup NEW 회원가입 및 분석 요청
▼
⚠️ 중요 (v1.3):
- shopInfo는 반드시 객체(Object) 형태로 중첩해서 보내야 합니다.
- category는 지정된 대문자 Enum 코드만 허용됩니다.

1. Request Body (JSON)

필드명 타입 필수 설명
emailString✅로그인 아이디 (이메일)
passwordString✅비밀번호 (8자 이상)
passwordConfirmString✅비밀번호 확인
nameString✅사장님 실명
phoneString휴대폰 번호
privacyAgreedBoolean✅개인정보 동의 (true)
shopInfo NEWObject✅가게 정보 객체
  .nameString✅가게 상호명
  .addressString✅가게 주소
  .categoryEnum✅KOREAN, CHINESE, JAPANESE,
WESTERN, CAFE_DESSERT, BAR, ETC
  .customCategoryString⚠️ETC일 때만 입력 (그 외 null)

2. 예시 데이터 (Example)

{
  "email": "owner@pulse.com",
  "password": "strongPass123!",
  "passwordConfirm": "strongPass123!",
  "name": "박사장",
  "phone": "010-1234-5678",
  "privacyAgreed": true,
  "shopInfo": {
    "name": "PULSE 치킨",
    "address": "서울시 강남구 테헤란로 123",
    "category": "CHICKEN",
    "customCategory": null
  }
}

3. Response

StatusData
201 CreatedaccessToken, message
400 Bad Request"비밀번호 불일치", "Enum 오류"
